#### 二、JVM内存结构
```bash
|-----------------------------------------------------------------------------|
|                         Class文件  => 类加载器                               |
|-----------------------------------------------------------------------------|
                                 ↑↓                                           |
|-----------------------------------------------------------------------------|
|                     运行时数据区（Runtime Data Area）                         |             
|-------------------------------------------|---------------------------------|
|                     |                     |                                 |
| 方法区（Method Area）| Java栈（Java Stack）| 本地方法区（Native Method Stack） |
|                     |                     |                                 |
|                     |                     |                                 |
----------------------|-------------------------------------------------------|
|                     |                                                       |
| 堆（Heap）           |           程序计数器（Progran Counter Register）       |    
|                     |                                                       |
|                     |                                                       |
|-----------------------------------------------------------------------------|
         ↑↓                          ↑↓
|---------------------|    |---------------------|    |-----------------------|
|      执行引擎        |    |       本地接口      |    |       本地库           |
| （Execution Engine）| -> | （Native Interface）| -> |  （Native Libranies）  |
|---------------------|    |---------------------|    |-----------------------|
```
